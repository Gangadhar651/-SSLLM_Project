# 🚀 Smart Skill & Live Learning Module (SSLLM)

## 📌 Overview
SSLLM is an AI-powered learning and project management platform designed to bridge the gap between academic learning and real-world industry requirements.

## 🎯 Features
- 📊 Skill Requirement Dashboard
- 🎥 Live Lecture System with Attendance Tracking
- 📁 Real-world Project Assignment
- 📈 Progress Tracking Dashboard
- 🤖 AI Learning Recommendations
- 🏆 Certificate Generation System

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP

## ▶️ How to Run
1. Install XAMPP
2. Place project in `htdocs`
3. Import database in phpMyAdmin
4. Start Apache & MySQL
5. Open:
   http://localhost/SSLLM_Full_Project
